C++

Introduction

Cet programme affiche "Hello World!" au flux de sortie standard:

#include < iostream>
int main() { std::cout << "Hello World!" << std::endl;
}

Analyse

On va examiner chaque partie de ce code en detail:
#include < iostream> est l’adresse préprocesseur (preprocessor directive) qui contient le fichiers entête de standard C++ iostream. iostream est un librairie standard fichier d’en-tête qui contiennent les définitions de l’entrée standard et sortie. Ces définitions sont comprises dans l’espace de nom « namespace » expliqué ci-dessous :

Les standard input /output (I/O) fournis le chemin à un programme pour avoir l’entrée et sortie à un système externe i.e. terminale. int main () { … } définie une nouvelle fonction nommée main. Par convention, la fonction main est appelée en haut de l’exécution des programmes. En programme C++, il y a exactement un seule fonction main, et doit retourner un nombre de type entier.

Par convention, le code de sortie 0 ou EXIT_SUCCESS d’un programme est interprété comme succès par un système qui exécute les programmes. Autre code de retourne est associé avec une erreur ; std ::cout << “ Hello World !” << std::endl; affiche “Hello World!” au flux de sortie standard : std est un espace de nom, et :: est le scope de l’opérateur de résolution qui permet de rechercher de l’objet par nom avec un espace de nom. std::cout est le flux de sortie standard, défini dans iostream, et il affiche à la sortie standard (stdout). << est, dans ce contexte, le l’opérateur de flux d’insertion, est appelé parce qu’il insère un objet dans le flux d’objet. La librairie standard défini le << operateur pour effectuer l’insertion des données pour certains type dans le flux de sortie.
